Logo Position Swap Navigation - Fixed

I've fixed the logo position swap functionality. Now the order correctly changes between collapsed and expanded states.

How the Fixed Logo Position Swap Works

When the navigation is in the collapsed state:

  • Logo image is displayed first (order: 1)
  • Then the "for" text (order: 2)
  • Then the website title "wnffldktm" (order: 3)

When the navigation is expanded:

  • The website title "wnffldktm" moves to the first position (order: 1)
  • Then the "for" text changes to "by" (order: 2)
  • Then the logo image (order: 3)

I also fixed the text swapping logic in JavaScript.

CSS Fix Details

The key changes:

/* Changed from row-reverse to row */
	.logo-container.expanded {
	flex-direction: row;
	}

	/* Explicit order definitions */
	.logo-container.expanded .logo-image-container { order: 3; }
	.logo-container.expanded .logo-text-container { order: 2; }
	.logo-container.expanded .logo-title { order: 1; }

JavaScript Fix

Fixed the text swapping logic:

// Changed from 'from' to 'by'
	logoText.textContent = isExpanding ? 'by' : 'for';
